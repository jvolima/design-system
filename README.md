# Design System
## O que é?
- Design System é um conjunto de componentes e tokens padronizados que podem ser usados em qualquer aplicação de uma empresa, desde textos, cores e botões.

## Tokens  do design system
- [x] Color
- [x] FontSize
- [x] FontWeight
- [x] Font
- [x] LineHeight
- [x] BorderRadius
- [x] Space
